# Sales Analytics and Forecasting Dashboard

## Project Overview
This project is an end-to-end Sales Analytics and Forecasting solution developed using Excel, SQL, Python, and Power BI. The project focuses on analyzing sales performance, identifying business insights, and forecasting future sales trends using machine learning techniques.

The dashboard provides interactive visualizations for revenue trends, regional performance, and category-wise sales analysis.

- SQL (MySQL)
- Python
- Power BI
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Project Workflow

### 1. Data Cleaning using Excel
- Removed duplicates
- Handled missing values
- Formatted date columns
- Created Month and Year columns
- Built Pivot Tables and Charts

### 2. SQL Analysis
Performed business analysis using SQL queries:
- Total Revenue Analysis
- Region-wise Sales
- Category-wise Sales
- Top Selling Products
- Monthly Sales Trends

### 3. Python Analysis & Forecasting
- Data analysis using Pandas
- Data visualization using Matplotlib
- Correlation analysis
- Sales forecasting using Linear Regression

### 4. Power BI Dashboard
Created an interactive dashboard containing:
- KPI Cards
- Monthly Revenue Trend
- Regional Sales Analysis
- Category-wise Revenue Distribution
- Interactive Filters/Slicers
